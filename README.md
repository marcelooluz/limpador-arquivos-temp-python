# Limpador de Arquivos Temporários

Projeto desenvolvido em Python para identificar, analisar e remover arquivos temporários do Windows.

## Funcionalidades

- Contagem de arquivos temporários
- Cálculo do espaço ocupado
- Listagem de arquivos
- Remoção de arquivos temporários
- Interface via terminal

## Tecnologias

- Python
- Módulo os

## Objetivo

Praticar automação, manipulação de arquivos e interação com o sistema operacional utilizando Python.
